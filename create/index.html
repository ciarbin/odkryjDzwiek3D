Digital Audio Workstations:
  • Sennheiser Orbit - Binaural AAX, VST, VST3 and AU plugins for both Mac and Windows.
  • Ambix - VST Ambisonic plugin suite by Matthias Kronlachner.
  • SPARTA - free VST plugin suite.
  • Ambi Pan - 3D panner for cinematic VR.
  • Facebook Spatial Workstation - A free software suite for designing spatial audio for 360 video and cinematic VR.
  • IEM Plugins Suite.
  • SPAT Gris - A free spatialization software for MacOS.
  • Wig Ware - free VST plugins for Ambisonics processing.
  • SoundField - Rode VST and AAX plugins for ambisonic processing.
Max/MSP:
  • SPAT - Max/MSP software suite for spatialization of sound signals in real-time intended for musical creation, postproduction, and live performances.
  • Jamoma for Max - Jamoma patches for Max/MSP.
  • Spatium - a set of free, open source and modular software tools for sound spatialization.
  • ICST Ambisonics tool - A set of externals for MaxMSP for Ambisonics surround sound processing and source-control in three dimensions.
Inne:
  • Envelope - Envelope software for Ableton Live.
  • Spatial audio - SoundScape renderer, standalone software for Linux and MacOS.
  • 3D Tune-In Toolkit - an open-source standard C++ library which includes a binaural spatialiser.
  • PyBinSim - binaural spatializer in Python.
